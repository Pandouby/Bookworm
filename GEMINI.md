# Bookworm

Bookworm is an iOS application built with SwiftUI, designed to be a comprehensive personal book management system. It empowers users to:

*   **Organize their library:** Add and keep track of books they own and have read.
*   **Track reading progress:** Assign ratings, record notes, and capture thoughts about specific books.
*   **Efficient book entry:** Conveniently add new books by scanning their barcodes.
*   **Personalized recommendations:** The application aims to leverage user ratings to provide intelligent recommendations for similar books, utilizing a multidimensional embedding space for advanced analysis.

## Core Technologies:

*   **SwiftUI:** For building a modern and declarative user interface across Apple platforms.
*   **GRDB:** A SQLite toolkit for Swift, used for robust and efficient local database management.
